This quick start is for platform operators.

## Summary

1. Define Core YAML in your infrastructure repo.
2. Review changes via PR.
3. Apply through your deployment pipeline.

For most developers, Core is managed by your platform team. Use [Blazing Flow](/docs/flow-overview) for application development.
