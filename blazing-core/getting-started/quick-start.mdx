## Prerequisites

- A Digital Frontier account
- Core CLI installed

## Installation

### macOS

```bash
brew install blazing-cli
```

### Linux

```bash
curl -fsSL https://get.digitalfrontier.so/core | sh
```

### Verify Installation

```bash
core --version
```

## Initialize Your Project

Create a new project directory:

```bash
mkdir my-app && cd my-app
core init
```

This creates a basic `compose.yaml` file.

## Your First Service

Edit `compose.yaml` to define a simple web service:

```yaml
version: "0.2"

services:
  web:
    image: nginx:latest
    cpu: 1
    memory: "2Gi"
    replicas: 2

    network:
      ingress:
        expose:
          - port: 80
            protocol: http
```

## Validate Your Configuration

Check for errors:

```bash
core validate
```

You should see:

```
✓ Schema validation passed
✓ Feature validation passed
✓ No compliance issues
```

## Preview the Deployment

See what will be deployed:

```bash
core plan
```

Output:

```

  Primary: GCP (cost-optimized)
  Fallback: [Akash, DFC]

Resources:
  ✓ Service: web (2 replicas)
  ✓ Load Balancer: HTTP (port 80)
  ✓ Auto-scaling: disabled
```

## Deploy

Apply your configuration:

```bash
core apply
```

Output:

```
⚙ Deploying...
  ✓ Creating service: web
  ✓ Provisioning load balancer
  ✓ Starting replicas (2/2)

✅ Deployment successful

Service URL: http://web-a3d5.df.app
Status: https://console.digitalfrontier.so/deployments/df-20251203-00123
```

## Check Status

View your deployment:

```bash
core status
```

## Update Your Service

Let's add HTTPS and autoscaling. Edit `compose.yaml`:

```yaml
version: "0.2"

services:
  web:
    image: nginx:latest
    cpu: 1
    memory: "2Gi"

    # Add autoscaling
    autoscaling:
      enabled: true
      min_replicas: 2
      max_replicas: 10
      target_cpu_percent: 70

    network:
      ingress:
        expose:
          - port: 443
            protocol: https
            tls:
              auto: true  # Automatic TLS certificates
```

Apply the changes:

```bash
core apply
```

Blazing Core will:
1. Provision TLS certificates automatically
2. Update the load balancer to HTTPS
3. Enable autoscaling (2-10 replicas)

## Add a Database

Let's add a PostgreSQL database. Update `compose.yaml`:

```yaml
version: "0.2"

services:
  web:
    image: nginx:latest
    cpu: 1
    memory: "2Gi"
    autoscaling:
      enabled: true
      min_replicas: 2
      max_replicas: 10
      target_cpu_percent: 70
    network:
      ingress:
        expose:
          - port: 443
            protocol: https
            tls:
              auto: true
    # Reference database credentials
    secrets:
      - secret_ref: db-credentials
        mode: env
        keys:
          host: DB_HOST
          username: DB_USER
          password: DB_PASSWORD

  db:
    image: postgres:15
    cpu: 2
    memory: "4Gi"
    replicas: 1
    storage:
      - mount_path: /var/lib/postgresql/data
        size: "50Gi"
        persistent: true
    network:
      ingress:
        expose:
          - port: 5432
            protocol: tcp
            internal_only: true  # Not exposed to internet
```

Create the database secret:

```bash
core secrets create db-credentials \
  --type db-credentials \
  --data host=db.internal \
  --data username=appuser \
  --data password=your-secure-password
```

Apply:

```bash
core apply
```

## Clean Up

Remove all resources:

```bash
core destroy
```

Confirm when prompted.
