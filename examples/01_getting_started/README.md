# Getting Started Examples

Simple examples to help you get started with Blazing quickly.

## Examples

### [hello_world.py](./hello_world.py)
Your first Blazing Core function. Learn the basics of creating and deploying serverless functions.

**Difficulty**: Beginner | **Time**: 5 min

### [basic_task.py](./basic_task.py)
Create and execute distributed tasks with Blazing Flow.

**Difficulty**: Beginner | **Time**: 10 min

### [parallel_execution.py](./parallel_execution.py)
Execute multiple tasks concurrently for maximum throughput.

**Difficulty**: Intermediate | **Time**: 15 min

## Running These Examples

1. Install Blazing:
```bash
pip install blazing
```

2. Configure your credentials:
```bash
blazing configure
```

3. Run an example:
```bash
python hello_world.py
```

4. Deploy to production:
```bash
blazing deploy hello_world.py
```
