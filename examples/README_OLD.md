# Blazing Examples

A collection of examples demonstrating various use cases and patterns.

## Getting Started

Browse the examples below by category. Each example includes its own documentation and can be run independently.

## Categories

### 01_getting_started
Basic examples to help you get started quickly.

### 02_web_endpoints
Examples of web servers, APIs, and HTTP endpoints.

### 03_data_processing
Data transformation, ETL, and batch processing examples.

### 04_async_parallel
Asynchronous programming and parallel execution patterns.

### 05_integrations
Integration examples with popular services and tools.

### 06_advanced
Advanced patterns and techniques.

## Running Examples

Each example can be run independently. Navigate to the specific example directory and follow the instructions in its README.
