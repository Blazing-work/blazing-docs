This guide shows how to run a sandboxed step and safely call a trusted service.

## 1. Define a Service

```python
from blazing.base import BaseService

@app.service
class DataService(BaseService):
    async def clean(self, data: dict) -> dict:
        return {"clean": True, **data}
```

## 2. Define a Sandboxed Step

```python
@app.step(sandboxed=True)
async def user_transform(data: dict, services=None):
    return await services["DataService"].clean(data)
```

## 3. Run It

```python
result = await app.user_transform({"value": 1}).wait_result()
print(result)
```

## Notes

- Sandboxed steps have restricted network and filesystem access.
- All external access must go through trusted services.
