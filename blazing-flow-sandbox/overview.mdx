Blazing Flow Sandbox lets you run **untrusted user code** safely while keeping your trusted services and credentials protected.

## What It Is

- User code runs in a **WASM/Pyodide** sandbox.
- Trusted services run outside the sandbox.
- User code can only interact with the outside world through approved service methods.

## When to Use

- End users write custom transforms.
- You run multi‑tenant workloads.
- You need strong isolation guarantees.

## How It Works (High Level)

1. User code executes in a restricted runtime.
2. Service calls are forwarded to trusted workers.
3. Results flow back to the sandboxed step.
