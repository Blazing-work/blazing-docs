Flow Sandbox provides a safe environment for running untrusted code.

## Core Capabilities

- Isolated execution
- Safe service bridge to trusted systems
- Optional checkpointing and session recovery
- Predictable resource limits

## How to Use

Mark steps or workflows as `sandboxed=True` and move sensitive logic into services.
