Flow Sandbox provides strong isolation for untrusted code while preserving access to trusted services.

## Key Capabilities

- WASM isolation for user‑provided code
- Explicit service bridge to trusted systems
- Optional checkpointing for session recovery
- Predictable resource limits and quotas

## Why It Matters

- Protects your infrastructure from user code
- Makes multi‑tenant workloads safe by default
- Keeps platform controls centralized in Blazing Core
