This quick start shows a minimal sandboxed workflow.

```python
import asyncio
from blazing import Blazing

async def main():
    app = Blazing()

    @app.step(sandboxed=True)
    async def user_step(x: int, services=None):
        return x * 2

    @app.workflow
    async def user_flow(x: int, services=None):
        return await user_step(x, services=services)

    await app.publish()
    print(await app.user_flow(3).wait_result())

if __name__ == "__main__":
    asyncio.run(main())
```

Use services when you need external access.
