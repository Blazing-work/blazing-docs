Sandboxed steps cannot access external systems directly. Services provide a safe bridge.

## Pattern

1. Define a service that wraps external access.
2. Call that service from a sandboxed step.

```python
from blazing.base import BaseService

@app.service
class UserService(BaseService):
    async def get_user(self, user_id: int) -> dict:
        return {"id": user_id}

@app.step(sandboxed=True)
async def user_profile(user_id: int, services=None):
    return await services["UserService"].get_user(user_id)
```

## Best Practices

- Keep services small and explicit.
- Validate inputs inside services.
- Avoid exposing raw credentials or unrestricted queries.
