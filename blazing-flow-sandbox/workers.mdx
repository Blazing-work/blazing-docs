Blazing Flow routes work to trusted or sandboxed workers automatically.

## Trusted Workers

- Run your services and workflow orchestration.
- Have network and filesystem access.

## Sandboxed Workers

- Run user‑provided code in a restricted runtime.
- No direct network or filesystem access.

You do not need to configure routing manually.
