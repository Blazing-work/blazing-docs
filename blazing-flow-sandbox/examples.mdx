## Example: User Transform

```python
@app.step(sandboxed=True)
async def user_transform(data: dict, services=None):
    return {"value": data["value"] * 2}
```

## Example: Service Bridge

```python
@app.service
class DataService(BaseService):
    async def enrich(self, data: dict) -> dict:
        return {"enriched": True, **data}

@app.step(sandboxed=True)
async def user_enrich(data: dict, services=None):
    return await services["DataService"].enrich(data)
```
