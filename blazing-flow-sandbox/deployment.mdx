Sandboxed execution is delivered as a **managed capability**. Your platform team configures the runtime through Blazing Core; developers just mark steps as `sandboxed=True`.

## What the Platform Provides

- Isolated runtimes for untrusted code
- Managed scaling and lifecycle
- Secure routing between sandboxed steps and trusted services

## What Developers Do

- Use `sandboxed=True` for user code
- Keep external access in services
