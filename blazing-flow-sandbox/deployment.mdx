Sandboxed execution is managed by Blazing Core. You do not need to deploy or operate sandbox workers directly.

## What You Do

- Mark steps or workflows as `sandboxed=True`.
- Configure scaling policies in Blazing Core YAML if needed.

## What the Platform Does

- Runs sandboxed code in isolated workers.
- Routes service calls to trusted workers.
- Manages scaling and lifecycle.
