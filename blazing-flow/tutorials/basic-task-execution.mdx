This tutorial shows the minimal workflow execution loop.

## 1. Define a Step and Workflow

```python
import asyncio
from blazing import Blazing

async def main():
    app = Blazing()

    @app.step
    async def add(x: int, y: int, services=None):
        return x + y

    @app.workflow
    async def add_flow(x: int, y: int, services=None):
        return await add(x, y, services=services)

    await app.publish()

    result = await app.add_flow(2, 3).wait_result()
    print(result)

if __name__ == "__main__":
    asyncio.run(main())
```

## 2. Return Types

Return JSON‑serializable values: dicts, lists, strings, numbers, and booleans.
