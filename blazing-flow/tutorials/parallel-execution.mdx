Parallelize work by launching multiple steps and awaiting them together.

## Fan‑Out / Fan‑In

```python
import asyncio
from blazing import Blazing

async def main():
    app = Blazing()

    @app.step
    async def square(x: int, services=None):
        return x * x

    @app.workflow
    async def squares(nums: list[int], services=None):
        runs = [square(n, services=services) for n in nums]
        results = await asyncio.gather(*[r.result() for r in runs])
        return results

    await app.publish()
    print(await app.squares([1, 2, 3, 4]).wait_result())

if __name__ == "__main__":
    asyncio.run(main())
```

## Best Practices

- Keep steps small and idempotent.
- Prefer fan‑out for I/O‑bound workloads.
