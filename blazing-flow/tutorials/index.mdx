## Quick Reference

### Task Lifecycle

```python
from blazing import Blazing

# 1. Initialize
app = Blazing()

# 2. Create task
unit = await app.create_workflow_task("workflow_name", param=value)

# 3. Execute & gather results
results = await app.gather([unit])

# 4. Cleanup
await app.close()
```

### Parallel Pattern

```python
# Create all tasks first
tasks = [
    await app.create_workflow_task("workflow", id=i)
    for i in range(100)
]

# Execute in parallel
results = await app.gather(tasks)
```

### Error Handling Pattern

```python
import asyncio

try:
    unit = await app.create_workflow_task("workflow", param=value)
    result = await asyncio.wait_for(
        unit.result(),
        timeout=30
    )
except asyncio.TimeoutError:
    print("Task timed out")
except Exception as e:
    print(f"Task failed: {e}")
```
