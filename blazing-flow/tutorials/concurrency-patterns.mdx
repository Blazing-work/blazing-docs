This guide shows simple concurrency patterns using standard `asyncio`.

## Bounded Concurrency

```python
import asyncio
from blazing import Blazing

async def main():
    app = Blazing()

    @app.step
    async def fetch(url: str, services=None):
        return url

    @app.workflow
    async def fetch_all(urls: list[str], services=None):
        sem = asyncio.Semaphore(5)

        async def limited(u: str):
            async with sem:
                run = await fetch(u, services=services)
                return await run.result()

        return await asyncio.gather(*[limited(u) for u in urls])

    await app.publish()

if __name__ == "__main__":
    asyncio.run(main())
```

## Best Practices

- Keep concurrency limits in workflows, not in steps.
- Tune limits based on downstream services.
